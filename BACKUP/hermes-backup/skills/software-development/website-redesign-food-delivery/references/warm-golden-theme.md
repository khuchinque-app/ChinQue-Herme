# Warm Golden Theme Reference — Tittil

## Design Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--tittil-accent` | `#FFC700` | Primary CTA, badges, active states |
| `--tittil-dark` | `#1A1A1A` | Primary text |
| `--tittil-card` | `#FFFFFF` | Card backgrounds |
| `--tittil-bg` | `#FFFBF5` | Warm cream background |
| `--tittil-glass` | `rgba(255,255,255,0.95)` | Header glass |

## Key Rules

1. Body: `background: #FFFBF5` warm cream
2. Hero: Forest green gradient with golden accent
3. Cards: White, `border-radius: 24px`, soft shadow
4. Add-to-cart: Black pill, white `+` icon
5. Floating cart: Golden yellow circle
6. Bottom nav: Pill-shaped white frosted glass
